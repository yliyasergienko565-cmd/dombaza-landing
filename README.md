# Домбаза — лендинг

Одностраничный лендинг магазина товаров для дома и ремонта.

**Живой сайт:** https://yliyasergienko565-cmd.github.io/dombaza-landing/

## Что на странице

- Hero с фото товаров и слоганом
- Четыре категории (инструменты, сантехника, электрика, отделочные материалы) с фото и тегами подкатегорий
- Блок «Почему мы»
- Интерактивный калькулятор доставки — выбор зоны (Москва и область / Санкт-Петербург / остальная Россия) показывает цену и срок
- Форма заявки (имя, телефон, что заказать)

## Технологии

Один файл `index.html` — HTML/CSS/JS без сборщиков и зависимостей. Шрифты Google Fonts (Fredoka + Nunito) через `<link>`.

## Деплой

Публикуется на GitHub Pages из ветки `main`, папка `/` (корень). Любой пуш в `main` автоматически обновляет живой сайт.

## Локальный просмотр

Просто откройте `index.html` в браузере — сборка не требуется.
